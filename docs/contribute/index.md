TVM 是由社区成员开发的，欢迎大家为 TVM 项目做出贡献。我们重视所有形式的贡献，包括但不限于：

- 现有补丁的代码 review
- 文档和使用示例
- 社区参与论坛和 issue
- 代码的可读性和开发手册
  - 为代码添加注释来提高可读性
  - 贡献文档来解释内部的设计选择
- 测试用例，使代码库更加稳健
- 推广项目的教程、博文和讲座
